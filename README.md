## Как запустить проект?

Для начала, необходимо запустить сервер разработки, введя в консоль:

```bash
npm run dev
# или
yarn dev
# или
pnpm dev
# или
bun dev
```

Откройте [http://localhost:3000](http://localhost:3000) в вашем браузере, чтобы увидеть результат.
